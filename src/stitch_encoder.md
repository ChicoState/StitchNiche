# A stitch can have its front and back (inverse). We show this with negation.
For example , the reverse of a knit stitch is perl, and vice versa. 
|  Stitch Name  | Stitch Code |
| ------------- | ------------- | 
|  No Stitch (empty)  | 0 | 
|  Knit / Purl  | 1 / -1 |
